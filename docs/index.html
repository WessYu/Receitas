Projeto de portfolio

Receitas full-stack com visual editorial.

Uma plataforma moderna feita com Next.js, TypeScript, Tailwind, Prisma, PostgreSQL, autenticacao, favoritos, dashboard privado e painel administrativo.

Ver repositorio Rodar localmente

Stack

Produto

Home, biblioteca filtravel, pagina individual, favoritos e perfil.

Admin

CRUD de receitas, categorias, usuarios e status de publicacao.

Seguranca

Senha com hash, cookie assinado, middleware e validacao Zod.

Banco

Schema Prisma com seed inicial e suporte a PostgreSQL.

Esta pagina no GitHub Pages e uma vitrine estatica. O app completo precisa de Node.js e PostgreSQL, por isso deve ser executado localmente ou em uma hospedagem full-stack.
